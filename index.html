Vinculación de un site con una base de datos - PHP

En este OVI encontrará los pasos y principios básicos entre la vinculación de un site desarrollado en lenguaje de programación PHP y un motor de bases de datos Mysql.

¿Qué aprenderé en este OVI?

• Entenderá el funcionamiento algunos de los sitios web más reconocidos así como también será capaz de comenzar y realizar su propio proyecto web.
• Aprenderá a utilizar uno de los gestores de desarrollo web, más completos como Adobe Dreamweaver, con el cual podrá realizar sitios web fácilmente y de manera rápida.
• Aprenderá cómo instalar un servidor de pruebas local como xampp, configurar un nuevo sitio, basado en lenguajes de programación PHP, HTML y hojas de estilos CSS, como también el uso y puesta en marcha de site que trabajen con bases de datos.

¿Qué son los estilos CSS?

Los estilos CSS son las instrucciones que lee cualquier navegador web y que definen la forma en que deben representarse todos los elementos que componen la página o el sitio que hemos realizado. CCS es la abreviatura de Cascading Style Sheets (en español Hojas de Estilo en Cascada). Mediante estos estilos CSS, el creador o desarrollador de la página puede definir color, fuentes, posición, alineación de imágenes o cualquier otro elemento.

HTML5 es la última versión de HTML, ha sido diseñado para ser utilizable por todos los desarrolladores de Open Web, este término es representado en dos conceptos: 1. Se trata de una nueva versión de HTML, con nuevos elementos, atributos y comportamientos. 2. Contiene un conjunto más amplio de tecnologías que permite a los sitios Web y a las aplicaciones ser más diversas y de gran alcance.
