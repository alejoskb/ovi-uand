Para que se utiliza Javascript

JavaScript es un dialecto de programación web descifrado. Es decir, los (contenidos) se ejecutan desde el lado del cliente y no desde el servidor, el código fuente para su ejecución a través de un procedimiento de recopilación (a pesar de que hay además un uso de JavaScript para las aplicaciones del lado del servidor, aún de uso menos sucesivo). Los contenidos son pequeñas actividades, impactos u ocasiones, que pueden combinarse para crear proyectos más grandes. Es habitual ver pequeños avances en JavaScript como artilugios: una gran parte de los cronometradores, los mapas dinámicos que demuestran su comienzo de la tierra, los minichats, los observadores de diapositivas y las máquinas de sumar que encontramos en este dialecto.
Los componentes tan básicos como un marcador de la fecha o nuestra información de ruta (IP, marco de trabajo, etc.) o los impactos, por ejemplo, que muestran una expresión irregular cada vez que volvemos a una página, son, además, promedio en JavaScript. Es básico para los impactos visuales, por ejemplo, cambiar el lapso de los componentes de la página o hacer avances vivificados entre las ilustraciones. La proliferación de sonido y video, las asociaciones entre la página y la cámara web o la boquilla del cliente, además de los paisajes de actividad de JavaScript. Entre los avances más importantes, las salas de visitas son normales, a pesar del hecho de que hay individuos que desarrollan videojuegos completos. JavaScript se mantiene en todas las etapas (aunque se pueden introducir nuevos elementos adicionales de vez en cuando) y no se requieren licencias para su utilización.

javaScript es un código de programación hecho por Netscape con el objetivo de incorporarlo a HTML y fomentar la producción de páginas inteligentes sin la necesidad de utilizar contenidos CGI o Java. Es fundamental no cometer errores JavaScript para Java. Java es un dialecto de programación que tiene un compilador, por lo que puede crear tanto aplicaciones como aplicaciones que se incorporan en la página HTML (applets). JavaScript es un codigo de contenido descifrado (no ordenado) que se carga como un aumento de HTML.
El código de un programa fuente JavaScript está incrustado directamente en el registro HTML y no debe incorporarse, es simplemente el programa que se encarga de descifrar el código. JavaScript es un código de programación basado en protestas, destinado a la mejora de las aplicaciones cliente / servidor a través de Internet. JavaScript le permite crear programas que se ejecutan específicamente en el programa (cliente) con el objetivo de que puedan llegar a tareas específicas o acercarse al servidor.
Es importante conocer a fondo estos tipos de conceptos para que estemos bien informados en el momento de seguir adelante con el curso de diseño de sitios web, esto mejorara nuestra comprencion de las tematicas
