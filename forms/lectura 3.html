Para que se utiliza Javascript

Un lenguaje occidental como el nuestro se lee de izquierda a derecha. En la medida que vamos avanzando en la lectura, se va añadiendo significado y se van formando, en nuestro cerebro, los conceptos relacionados entre si. --------> Entonces tenemos dos puntos de vista del codigo de programacion, unos escriben, otros lee. La forma de representarlo seguira la siguiente Nosotros escribimos Las maquinas leen Navegarores capaces de leer el codigo

Es importante conocer a fondo estos tipos de conceptos para que estemos bien informados en el momento de seguir adelante con el curso de diseño de sitios web, esto mejorara nuestra comprencion de las tematicas javaScript es un código de programación hecho por Netscape con el objetivo de incorporarlo a HTML y fomentar la producción de páginas inteligentes sin la necesidad de utilizar contenidos CGI o Java. Es fundamental no cometer errores JavaScript para Java. Java es un dialecto de programación que tiene un compilador, por lo que puede crear tanto aplicaciones como aplicaciones que se incorporan en la página HTML (applets). JavaScript es un codigo de contenido descifrado (no ordenado) que se carga como un aumento de HTML.El código de un programa fuente JavaScript está incrustado directamente en el registro HTML y no debe incorporarse, es simplemente el programa que se encarga de descifrar el código. JavaScript es un código de programación basado en protestas, destinado a la mejora de las aplicaciones cliente / servidor a través de Internet. JavaScript le permite crear programas que se ejecutan específicamente en el programa (cliente) con el objetivo de que puedan llegar a tareas específicas o acercarse al servidor./
